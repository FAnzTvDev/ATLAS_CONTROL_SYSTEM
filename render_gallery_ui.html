Autonomous Workflow

Use this checklist to move a scene from script ➝ manifest ➝ pipeline ➝ render ➝ QA without leaving the UI.

  1. Select a director profile and regenerate the shot list for the manifest.
  2. Run the Pipeline Agent to validate assets, build the 14-slot grid, and emit the shot plan.
  3. Fire the Renderer Agent to send each shot to Fal Nano Banana Pro with the locked refs.
  4. Review QA Agent results (Vision + script rules) before approving or triggering reruns.

All four agents write logs to the statuses below so you can watch progress in real time or hand the steps off to an automated worker.
