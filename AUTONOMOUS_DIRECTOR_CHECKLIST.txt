🏗️  ARCHITECTURE SUMMARY:

Entry point:     async run_autonomous_scene(project, scene_id, project_path, options, progress_callback)
Main class:      AutonomousDirectorAgent(project, scene_id, project_path)

9-stage pipeline:
  1. check_preflight()                   → verify shot_plan.json, cast_map.json, story_bible.json
  2. fix_dialogue_distribution()         → inject "character speaks:" markers into LTX
  3. generate_multi_angle_variants()     → 3 Qwen variants per shot (turbo parallel)
  4. auto_select_best_variants()         → LOA scoring (identity 60%, location 20%, CLIP 10%, comp 10%)
  5. render_scene_videos()               → LTX-2 image-to-video (turbo parallel)
  6. stitch_extended_segments()          → >20s shots: segment_0.mp4 + segment_1.mp4 → full.mp4
  7. stitch_full_scene()                 → FFmpeg concat all shots into final render
  8. generate_comparison_grids()         → 3-variant side-by-side still frames per shot
  9. generate_report()                   → cost breakdown, timing, success metrics


💰 COST MODEL (per-scene):

First frames:        $0.08/shot
Multi-angle variants: $0.12/variant × 3 = $0.36/shot
Video rendering:     $0.08/shot
Segment stitching:   $0.02/pair
Scene stitch:        $0.01 (one-time)
Comparison grids:    $0.03/shot

Example (20-shot scene):
  20 × ($0.08 + $0.36 + $0.08 + $0.03) + $0.12 + $0.01 = $11.13

🔌 API ENDPOINTS:

1. Launch autonomous render
   POST /api/v18/autonomous/render-scene
   Request:  {project, scene_id, dry_run, options}
   Response: {status, cost, timing, output_files, errors}

2. Check render status
   GET /api/v18/autonomous/status/{project}/{scene_id}
   Response: {status, output_file, file_size_gb, exists}

3. Estimate cost upfront
   POST /api/v18/autonomous/estimate-cost
   Request:  {project, scene_id}
   Response: {cost_estimate, cost_breakdown, timing_estimate}
